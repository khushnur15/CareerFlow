# 🚀 CareerFlow

> AI-Powered Career Intelligence Platform built using React, TypeScript, Tailwind CSS, FastAPI, and Python.

CareerFlow helps users analyze resumes, identify skills, discover matching job opportunities, and generate personalized learning roadmaps.

---

# ✨ Features

- 📄 Resume Upload
- 🧠 Skill Extraction
- 💼 AI Job Matching
- 🎯 Skill Gap Analysis
- 🗺 Personalized Learning Roadmap
- 📊 Career Intelligence Dashboard
- ⚡ FastAPI Backend
- 🎨 Modern React Frontend

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/khushnur15/CareerFlow.git
```

## Backend Setup

```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python -m uvicorn app.main:app
```

Backend runs at:

```text
http://127.0.0.1:8000
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at:

```text
http://localhost:5173
```
